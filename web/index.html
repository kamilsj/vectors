START IN FIVE MINUTES

Relational filters meet vector similarity.

vectors keeps the workflow deliberately familiar: define a schema, insert embeddings, and query everything with SQL.

SELECT id, title,
  cosine_distance(
    embedding,
    ARRAY[1, 0, 0]
  ) AS distance
FROM documents
WHERE category = 'tech'
ORDER BY distance
LIMIT 5;
